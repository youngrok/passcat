# 패스캣

패스캣은 공동인증서를 관리하고 공동인증서가 필요한 여러 가지 웹사이트에 자동으로 로그인해주는 데스크탑 앱입니다. 지원되는 웹사이트는 다음과 같습니다.

- 홈택스 https://hometax.go.kr/
- 위택스 https://wetax.go.kr/
- 사회보험 https://si4n.nhis.or.kr/
- 비씨카드, 은행 등은 개발 예정

## 다운로드

최신 버전은 [Releases](https://github.com/youngrok/passcat/releases/latest) 페이지에서 받을 수 있습니다.

- **Windows**: `passcat-<버전>-win-setup.exe`
- **macOS**: `passcat-<버전>-mac.dmg`

각 설치 파일 옆의 `.sha256`와 `.sig`는 무결성·서명 확인용입니다. 앱 내부의 자동 업데이트 기능이 이 파일들을 이용해 다운로드한 설치 파일을 검증합니다.

## 보안 경고
아직 패스캣은 데스크탑 앱 개발에 필요한 코드 서명 인증서를 구입하지 않아서 앱을 실행하면 보안 경고가 뜰 수 있다는 점 양해 부탁드립니다. 다음과 같은 방법으로 보안 경고를 해제할 수 있습니다.

빠른 시일 내로 코드 서명 인증서를 구입해서 서명 절차를 거쳐서 보안 경고가 뜨지 않도록 조치하겠습니다.

## Windows 보안 경고

Windows에서 설치 파일을 실행하면 **"Windows의 PC 보호"** (SmartScreen) 경고가 나타날 수 있습니다. 다음과 같이 해제할 수 있습니다.

1. **"Windows의 PC 보호"** 창이 나타나면 **"추가 정보"**를 클릭합니다.
2. 하단에 **"실행"** 버튼이 나타나면 클릭하여 설치를 진행합니다.

## macOS 보안 경고

macOS에서 처음 실행할 때 **"확인되지 않은 개발자"** 경고가 나타날 수 있습니다. 다음과 같이 해제할 수 있습니다.

1. DMG를 열고 앱을 Applications 폴더로 드래그합니다.
2. 앱을 **우클릭(또는 Control+클릭)** → **"열기"**를 선택합니다.
3. 경고 대화상자에서 **"열기"**를 클릭합니다.
4. 이후에는 정상적으로 실행됩니다.

## 자동 업데이트

설치 후 앱을 실행하면 시작할 때마다 새 버전이 있는지 확인합니다. 업데이트가 있으면 화면 상단에 안내가 표시되고, 버튼을 누르면 앱이 자동으로 종료·재설치·재실행됩니다. 사용자 데이터(등록한 인증서·로그·설정)는 그대로 유지됩니다.
